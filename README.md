## 🌍 About EcoTrack

EcoTrack is an **AI-powered carbon footprint tracking platform** that helps individuals monitor their daily activities, calculate personal emissions, and take meaningful action toward a sustainable lifestyle.

With region-aware data, an intelligent AI assistant, gamified eco-challenges, and a verified offset marketplace, EcoTrack makes going green accessible, engaging, and impactful.

---

## ✨ Features

### 🧮 Carbon Calculator
- Multi-step footprint calculator covering **transport, home energy, and diet**
- Uses **region-specific emission factors** (supports India and other countries)
- AI-generated personalized insights after calculation

### 🤖 AI Climate Assistant
- Powered by **Llama 3** — ask anything about carbon, climate, and sustainability
- Region-aware responses tailored to your local grid, policies, and averages
- Suggested questions to get you started instantly

### 📊 Personal Dashboard
- Track your footprint **month by month** with real-time charts
- Compare against national and Paris Agreement targets
- Visual breakdown by category: transport, energy, food, shopping, home

### 🌍 Planet Intelligence
- Global CO₂ monitoring with **AI trend forecasting**
- 12-month emissions projection with high-confidence modeling
- Regional hotspot detection and top emitter breakdowns

### 🌳 Offset Marketplace
- Verified, science-backed projects (reforestation, clean energy, ocean)
- AI-matched recommendations based on your region and profile
- Examples: **Western Ghats Reforestation**, **Rural Solar India**

### 🏆 Eco Challenges & Gamification
- Real-world challenges: Zero-Car Week, Plant-Based Month, Energy Fast
- Earn **XP, badges, and streaks** for completing actions
- AI verification of challenge progress

### 🥇 Community Leaderboard
- Global and city-level rankings
- Live activity feed from the EcoTrack community
- Top performers earn exclusive rewards

### 📚 Learn Hub
- Evidence-based guides, infographics, and research articles
- Topics: Carbon Footprint 101, Aviation & Climate, Food & Diet, and more

## 🏃 Running Locally

No build tools or dependencies required!

```bash
# 1. Clone the repository
git clone https://github.com/Hardin-17/ecotrack.git

# 2. Navigate into the folder
cd ecotrack

# 3. Open in your browser
open index.html
# or just drag index.html into any browser
```

## 🌐 Region Support

EcoTrack currently defaults to **🇮🇳 India** with emission factors calibrated for:
- India's electricity grid intensity
- Local transport averages
- National dietary patterns
- India-specific offset projects (Western Ghats, Rural Solar)

More regions can be selected via the region switcher in the app.
